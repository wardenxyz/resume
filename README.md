# Typst 简历模板（自用）

个人自用简历模板，带头像，使用 Typst 编写，魔改自 https://github.com/OrangeX4/Chinese-Resume-in-Typst

我改了四个版本，根目录下的 [Profile.jpg](/profile.jpg) 是头像

简历包含

- 姓名
- 专业技能
- 教育背景
- 项目经历
- 校园经历
- 实习经历
- 自我评价
